Spark

Java, XML

Spark is a music discovery app I made using Android Studio and Java where users could play songs and swipe left or right to like or dislike them respectively. Swiping up would let the user share the song via SMS. I used the Spotify SDK to play music and implemented automatic account authentication. I then used a touch listener to track the user's swipes, and performed the cooresponding action based on which direction they swiped. The left and right swipes actions were done with the SDK, but when the user swiped up they had to be able to share the current song. To do this, I used Android's SMS manager to send a link to the current song to the specified phone number.
